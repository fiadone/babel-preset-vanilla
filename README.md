# @fiad/babel-preset-vanilla

A custom *Babel* preset for vanilla JS development

---

## Installation

```
npm i @fiad/babel-preset-vanilla
```

## Usage


```json
// babel.config.json
{
  "presets": ["@fiad/vanilla"]
{
```

For more information, check the [*Babel* documentation](https://babeljs.io/docs/en/configuration).
